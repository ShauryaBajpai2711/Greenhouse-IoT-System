# 🌱 Greenhouse Monitoring and Control System using IoT

## 📌 Description
This project is an IoT-based greenhouse system designed to monitor and control environmental conditions like temperature, humidity, and soil moisture in real-time.

## 🎯 Objective
To automate greenhouse operations and improve plant growth by maintaining optimal environmental conditions.

## 🛠 Components Used
- Arduino UNO
- DHT11 Temperature & Humidity Sensor
- Soil Moisture Sensor
- Relay Module
- Water Pump
- WiFi Module (ESP8266)

## ⚙️ Features
- Real-time temperature and humidity monitoring
- Soil moisture detection
- Automatic irrigation system
- Remote monitoring using IoT
- Low-cost and efficient system

## 💻 Technologies Used
- Arduino (Embedded C)
- HTML, CSS, JavaScript
- IoT Platform (ThingSpeak / Blynk)

## 🚀 How It Works
1. Sensors collect environmental data
2. Arduino processes the data
3. Data is sent to IoT platform via WiFi
4. System automatically controls water pump based on soil moisture
